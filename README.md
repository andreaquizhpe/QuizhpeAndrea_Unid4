# Proyecto POO - Gestión de Contenido Audiovisual

1. Descripción del Proyecto
Este proyecto implementa un sistema de gestión de contenidos audiovisuales utilizando Programación Orientada a Objetos en Java.
Permite modelar diferentes tipos de contenido como películas, series de TV, documentales, podcasts y videos de YouTube, aplicando conceptos como herencia, asociación, agregación y composición.

---

2. Objetivos
- Aplicar conceptos de Programación Orientada a Objetos
- Implementar relaciones entre clases
- Simular un sistema de contenido audiovisual

---

3. Clases implementadas

3.1. Clases base
- ContenidoAudiovisual
- Pelicula
- SerieDeTV
- Documental

3.2. Clases adicionales
- Actor (Asociación con Pelicula)
- Temporada (Composición con SerieDeTV)
- Investigador (Agregación con Documental)
- Podcast
- VideoYoutube

---

3.3. Relaciones implementadas
- Asociación: Pelicula - Actor
- Agregación: Documental - Investigador
- Composición: SerieDeTV - Temporada
